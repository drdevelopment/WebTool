On top of the screen the menu 'Pages' appears. If you want to edit the page, please click on the menu 'Pages'. If you are lost in the dashboard, click on 'Dashboard' and you will return to this page.

 




It is time to open your website. If you click on the button 'Open website' a new tab in the browser is opened and your website is displayed.
